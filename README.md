## Environment:

NodeJS, RailsInstaller (Ruby 2.3, Rails 5), XAMPP

###### Server
* `$ bundle install`

* `$ rake db:create`

* `$ rake db:migrate`

* run: `$ rails s`

###### Client
* `$ bower install`

* `$ npm install`

* run: `$ http-server` or `$npm run start`

###### MySQL
* user: `root`
* password: none

## Dev Support Tool
###### Build sass, ES6
* `$ gulp es6:build`

* `$ gulp sass:build`

###### Build on saving
* `$ gulp js:watch`

* `$ gulp css:watch`
